Week 9 Code & Notes

JS Ninja Reading

BOM = Browser Object Model

Global variables are created without using const, let or var.

Dialogs

Browser dialogs are produced by 3 functions: alert(), confirm() and prompt(). Alert pops up a message, confirm asks for confirmation or cancelation, and prompt asks for input from user.

Browser information

userAgent - returns information about browser used. Not always accurate, can be spoofed. *deprecated*

Location Properties

  • location.href - returns info about current URL. It can also change the current URL.
  • location.protocol - describes protocol being used (http, https, ftp, etc.).
  • location.host - gives domain and port of current URL.
  • location.hostname - gives current domain.
  • location.port - gives current port
  • location.pathname - returns path after domain (e.g. /wk08/team).

  • location.search - returns query string (part of URL starting with '?').
  • location.hash - returns fragment identifier starting with '#'.
  • location.origin - shows protocol and domain of current URL.

Location Methods

  • reload() - forces reload of current page, either from cache or server.
  • assign() - loads resource from another URL
  • replace() - current page is not stored in session history eliminating ability to go back to it.
  • toString() - returns URL as a string

History

Gives information about previously visited pages

  • history.length - shows number of pages visited previous to current one
  • history.go() - lets you jump to a specific page in the history
  • history.forward() - move one page forward in history
  • history.back() - move one page back in history

Screen info

  • screen.height & screen.width - gives height and width of screen in pixels
  • screen.availHeight & screen.availWidth - gives height and width minus system menus (e.g. task bar)
  • screen.colorDepth - gives color depth of user monitor

Document Object

document.write() - writes string of text to page but will overwrite anything already there unless used inside script tags.

Cookies

Locally stored information about a browser session that can be accessed upon next visit to same domain. Used to personalize visit, authenticate user and keep track of choices. Cookies expire after session is done unless given an expiration date.

Timing functions

setTimeout() - delays the start of a function by a specified amount of time.

clearTimeout() - cancels above

setInterval() - repeats function by set time interval

clearInterval() - stops setInterval()

Timing can be used to animate page elements.
