Arrow functions do not have a "this" but instead reference the outer function

  let calculator = {
        read() {
          this.num1 = +prompt('1st number?');
          this.num2 = +prompt('2nd number?');
        },
    
        sum() {
          return this.num1 + this.num2;
        },
    
        mul() {
          return this.num1 * this.num2;
        }
    
      }
    
      calculator.read();
    alert( calculator.num1 + ' + ' + calculator.num2 + ' = ' + calculator.sum() );
    alert( calculator.num1 + ' * ' + calculator.num2 + ' = ' + calculator.mul() );

You Don't Know JS Reading

Statements = a group of words, numbers and operators that performs a specific task. (e.g. 

a = b * 2
)

Expressions = components of a statement. Using above example:

  • '2' is a literal value expression
  • 'b' is a variable expression
  • 'b * 2' is an arithmetic expression
  • 'a = b * 2' is an assignment expression
Other expressions:
  • Expression statement - 'b * 2'
  • Call expression - 'alert(a);'

Interpreting code = translation of code to computer language done at runtime.

Compiling code = translation of code to computer language done prior to runtime.

Javascript code is compiled then run at runtime.
