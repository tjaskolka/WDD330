HTML & CSS3 Reading

TRANSFORMS = allow you to translate (reposition), rotate, scale and/or skew BLOCK elements.

  • Translate = move up, down, left, right. Movement is relative to current position.
  • Scale = use a positive number to change size <1 will decrease, >1 will increase.
  • Rotate = turns element on an axis to a certain angle.
  • Skew = changes perspective on element to simulate movement in a 3D plane.

Origin = point of reference for transformation.

Order = transformations will be executed in the order they are listed, so having one before another will affect what comes after.

TRANSITIONS = provide a way for CSS values to change over time with the appearance of animation.

  • transition-property = defines the property(ies) that will be transitioned and has to have a midpoint.
  • transition-timing-function = controls the pace of the transition.
  • transition-delay = wait before starting transition

Animations

CSS animations allow fine control of animation via keyframes, allowing more changes in between the beginning and end state than a transition would. They are a light-weight alternative to JavaScript animations.

  • animation-name - used to associate an animation with an element
  • animation-duration - defines the length of time for one round of the animation.
  • animation-timing-function - defines the progress of the animation.
  • animation-iteration-count - specifies how many rounds the animation will play
  • animation-direction - which direction the animation will play (i.e. forwards or backwards).
  • animation-delay - time period before animation starts.
  • animation-fill-mode - defines what happens before animation begins and after it ends.
  • animation-play-state - defines state of animation (running or paused).

Canvas

Used to draw things using JavaScript. Can eliminate need to download images. It also manipulates pixels in images and video. Requires that a height and width be set to define the 'canvas' space. Canvas drawings can be saved. External images can be 'drawn' on the canvas, allowing pixel manipulation.
